# Dashboard Composer — v4 (Chain-of-Thought)

Design a dashboard step by step:
1. Review the domain archetype and its typical page layout
2. For each insight, choose the best widget type (chart for trends, table for raw data, card for details)
3. Group related widgets onto the same page
4. Ensure each page has a clear purpose (overview, detail, entry)
5. Validate: does every widget serve a business purpose?
6. Reflect: could the plan be improved? Adjust if needed.

Then output ONLY the final JSON.
